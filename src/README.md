# Mini-ToDO

一个仿照小米澎湃3笔记界面制作的的待办事项管理应用。

## 功能特性

- ✅ 添加待办事项
- ✅ 标记任务完成/未完成
- ✅ 自动保存数据（localStorage）
- ✅ 完成的任务自动移到底部
- ✅ 优雅的界面设计
- ✅ 响应式布局

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 使用方法

### 添加任务

1. 点击右下角的 `+` 按钮
2. 在输入框中输入待办事项
3. 按回车键确认

### 标记完成

点击任意任务即可标记为完成状态：
- 完成的任务会显示对号
- 任务文字会变暗并添加删除线
- 完成的任务会自动移到列表底部

### 自动管理

- 系统自动存储数据到本地

## 实现思路

### HTML
在主容器中生成输入框input，添加按钮button和代办任务列表ul作为组成网页的主要元素

### CSS
对HTML元素和JS文件中生成的元素进行设计，排版仿照小米澎湃3笔记界面，尽量简洁明了

### JS
在JS中实现除HTML生成元素功能外的其他元素及其功能，并绑定元素功能
- 任务列表实现逻辑：通过从input中获取的去除空格的待办内容，添加进列表list中
- 添加按钮逻辑：通过点击添加按钮切换生成输入框的show类，实现输入框的显示与隐藏；显示输入框后对齐绑定对回车键enter的事件监听，在按下回车键时添加任务并隐藏输入框
- 删除按钮逻辑：在每一次加入代办任务后生成单独的删除按钮，点击删除按钮后阻止事件冒泡名remove相关待办
- 完成任务切换逻辑：在每一次加入代办任务后生成单独的完成状态显示框，对待办栏绑定点击监听，在点击待办栏后切换待办文本的状态框的completed类，并将待办添加到列表底部

### 界面设计

- 黑色主题，简洁大方
- 黄色按钮，醒目易操作
