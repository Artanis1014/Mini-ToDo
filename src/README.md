# Mini-ToDO

一个仿照小米澎湃3笔记界面制作的的待办事项管理应用。

## 功能特性

- ✅ 添加待办事项
- ✅ 标记任务完成/未完成
- ✅ 自动保存数据（localStorage）
- ✅ 自动管理任务数量（最多保留10条）
- ✅ 完成的任务自动移到底部
- ✅ 优雅的界面设计
- ✅ 响应式布局

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 使用方法

### 添加任务

1. 点击右下角的 `+` 按钮
2. 在输入框中输入待办事项
3. 按回车键确认

### 标记完成

点击任意任务即可标记为完成状态：
- 完成的任务会显示对号
- 任务文字会变暗并添加删除线
- 完成的任务会自动移到列表底部

### 自动管理

- 系统自动存储数据到本地
